ID's vs Classes

What's Best Practice

7th September 2023

ID's and Classes are identifiers assigned to elements in an HTML document. Although they have similar functions, they are not the same and they do not always serve the same purpose.

Assigning Classes and ID's to HTML elements allows you to target groups of, or, specific elements with CSS Styling or JavaScript Functions.

A simple way to differenciate when you should use a class or an ID is to ask yourself "is there more than one element which requires the same style, now or at any time in the future?", and the answer is even "maybe", then make it a class. ID's are unique and classes are not.
